# Macro Attribution Dashboard 🎯

A Retrieval-Augmented Generation (RAG) powered cross-market decision dashboard designed to connect **US Macro trends**, **Web3 Real-World Asset (RWA) on-chain data**, and **Taiwan equity market spillover effects**.

## 🚀 Key Features

*   **Cross-Market Attribution**: Automated analysis of how Federal Reserve macro policies impact both US and Taiwan equity markets.
*   **Agentic Finance Data Flow**: Real-time tracking of Web3 RWA asset net inflows and autonomous rebalancing signals.
*   **VIX Fear Gauge**: Quantifies global hedging sentiment and market volatility to assist in risk assessment.
*   **Bilingual Interface**: Supports seamless real-time switching between Traditional Chinese (繁體中文) and English.
*   **Tech Stack**: Built with Python, TailwindCSS, and a RAG-based AI Engine.

## 🛠️ Deployment

1. **Fork** this repository.
2. Upload your `index.html` file to the root directory.
3. Navigate to **Settings -> Pages** in your repository and enable the GitHub Pages feature.
4. Your dashboard will be live and automatically accessible via your GitHub Pages URL.

## 📈 How to Update

To push new data or updates:
1. Generate your latest `index.html` file.
2. Upload and replace the existing `index.html` in your repository.
3. Commit the changes, and GitHub Pages will update your site automatically within seconds.

This dashboard is an experimental financial decision support tool. All attribution data is generated by AI models and is intended for **personal research purposes only**. Please perform your own due diligence before making any investment decisions.
