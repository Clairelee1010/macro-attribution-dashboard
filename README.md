# Macro Attribution Dashboard 🎯

一個基於 RAG (檢索增強生成) 技術的跨市場決策看板，專為串聯 **美股總經**、**Web3 RWA (真實世界資產) 鏈上數據** 與 **台股科技供應鏈溢出效應** 所設計。

## 核心功能 (Key Features)
- **跨市場聯動歸因**：自動化分析 Fed 宏觀政策對台美股的影響。
- **Agentic Finance 數據流**：追蹤 Web3 鏈上 RWA 資產淨流向。
- **VIX 恐慌追蹤器**：即時量化全球避險情緒水位。
- **雙語介面**：支援中/英雙語即時切換。
- **技術棧**：Python, TailwindCSS, RAG-based AI Engine。

## 部署說明 (Deployment)
1. Fork 此專案。
2. 將您的 `index.html` 放入根目錄。
3. 至 Settings -> Pages 開啟 GitHub Pages 功能。
4. 網址即會自動生成並全球同步。
